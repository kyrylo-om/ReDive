How to use

Homepage

1. Info button

Sends you to the info page

2. Head

Sends you to the homepage

3. Theme-toggle

Switches between dark (default) and light themes

4. Searchbar

Here the username that you would like to analyse should be typed in. You can begin to analyze by pressing the search-button on the right or by pressing Enter. Right afterards you will be sent to an analysis page. Note: if the username you typed is wrong or the username does not exist you will see an error message.

5. Database button

Forwards you to the database page. There you can see every account ever analyzed.

6. Contacts

Our support contacts. You can write an email to us just by clicking the link.

Database page

7. Sort options

By clicking you can see different sort options that you can apply by pressing an "apply" button.

8. Searchbar

Helps you to navigate the database. After pressing the search button only the users who has input text in the username will be shown.

9. Account card

Shows main info about a user: avatar, posts, comments and bot rank. If clicked, the deatiled analysis of the user will be shown.
